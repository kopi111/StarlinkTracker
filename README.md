# Starlink Device Tracker - Jamaica

Interactive web application for tracking Starlink satellite internet devices across Jamaica, designed for police stations and key locations.

## Features
- **Interactive Map**: Leaflet.js with heat maps and clustering
- **Device Management**: Full CRUD operations for Starlink devices
- **Geographic Filtering**: 14 Jamaican parishes (Kingston, St. Andrew, St. Catherine, etc.)
- **Location Types**: Police stations, schools, hospitals, government buildings
- **Status Tracking**: Active/Inactive device monitoring
- **GPS Coordinates**: Precise latitude/longitude tracking
- **Dashboard Analytics**: Statistics and device counts
- **Excel Export**: Device data with EPPlus library
- **GeoJSON Export**: Geographic data export
- **Red Zone Highlighting**: Special attention areas
- **Heat Map Visualization**: Device density mapping

## Technology Stack
- **Backend**: ASP.NET Core 8.0 MVC
- **Database**: SQLite with Entity Framework Core
- **Frontend**: Leaflet.js, Bootstrap 5, jQuery
- **Export**: EPPlus (Excel), GeoJSON
- **API**: Swagger/OpenAPI documentation

## Use Cases
- Network infrastructure monitoring
- Police station connectivity tracking
- Strategic device placement planning
- Coverage analysis
- Emergency response coordination
